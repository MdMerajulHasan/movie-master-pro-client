Overview: A website where people can explore movies, add movies, update their added movies etc.

Live link: https://silver-stardust-bda902.netlify.app/

Technologies: React Router, React.js, Tailwind, Firebase.

Core features: Explore and filter movies, add movie, edit movie, delete movie, login, register (google/email+password).

Dependencies:

Step-by-step guide on how to run the project locally:  1. clone the project 
                                                       2. open with vs code 
                                                       3. command: npm i 
                                                       4. add firebase to your project (optional)
                                                       5. npm run dev
